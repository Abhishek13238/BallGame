# 3D Ball Game (Unity)

This is a simple 3D Unity Ball Game demonstrating basic gameplay mechanics, player control, and level design.

## Features
- Collectible coins for scoring
- Obstacles to avoid
- 3 different levels with increasing difficulty

## Controls
- Arrow keys: Move the ball
